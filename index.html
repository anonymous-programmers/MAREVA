Multi-label Active REView classificAtion (MAREVA)

This is a mock page of the authors website to comply with the double-blind review of COMPSAC. The webpage containing the actual working version of this page will be available upon the publication of the paper.

This is a demonstration of MAREVA on automatically classifying user reviews into four classes i.e., Bug reports, Feature requests, User experiences and Rating. MAREVA's classification relies on a built logistic regression model using Multi-label Active Learning With Auxiliary Learner. 

As you input your recieved user review in the textbox below and press analyze. MAREVA uses NLTK for the review preprocessing and LIBACT to run the algorithms we mentioned above for the review classification. The output of the classification  shows the potential classes that your reviews belongs to.

Example

If the following message is used as input: Crashes please fix it for KitKat, it crashes when I open music player and it doesn’t scrobble. The output of the classification would identify this message as a Bug Report.
